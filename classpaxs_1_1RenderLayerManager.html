Detailed Description

レンダリングレイヤーマネージャー

Rendering layer manager

IRenderable オブジェクトを登録し、Z順序（レイヤー順）に従って自動的に描画します。 Registers IRenderable objects and automatically renders them in Z-order (layer order).
