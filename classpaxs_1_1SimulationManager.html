Detailed Description

シミュレーション管理クラス

Simulation Manager Class

SettlementSimulatorのファサードとして機能し、UIコンポーネントからの 直接アクセスを防ぐ。シミュレーションの初期化、実行、状態取得を管理する。 Acts as a facade for SettlementSimulator, preventing direct access from UI components. Manages simulation initialization, execution, and state retrieval.
