Detailed Description

Photo360入力ハンドラー

Photo360 input handler

Photo360Layerのキーボード入力を処理します。 Photo360が表示中（ViewMenu::view_3d）の場合のみ入力を処理し、 背後のレイヤー（Settlement、Map）への入力伝播をブロックします。 Handles keyboard input for Photo360Layer. Only processes input when Photo360 is visible (ViewMenu::view_3d), blocking input propagation to background layers (Settlement, Map).
