Detailed Description

UIパネルの背景と影を描画するクラス

Class for rendering UI panel background and shadow Siv3Dでは影を一度にまとめて描画することで、影の重複を防ぎます。 For Siv3D, renders all shadows at once to prevent shadow duplication.
