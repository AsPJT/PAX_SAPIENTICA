Detailed Description

描画可能オブジェクトの基底インターフェース

Base interface for renderable objects

このインターフェースを実装することで、オブジェクトはRenderLayerManagerによって Z順序に従って自動的に描画されるようになります。 By implementing this interface, objects can be automatically rendered in Z-order by the RenderLayerManager.
