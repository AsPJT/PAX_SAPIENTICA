Detailed Description

3Dモデル地物を表すクラス

Feature class representing a 3D model (burial chambers, buildings, etc.)

Note
将来の実装: 古墳の石室、建物などの3Dオブジェクトを地図上に配置
Future implementation: Place 3D objects like burial chambers, buildings on the map
