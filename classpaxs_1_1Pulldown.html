Detailed Description

プルダウンメニューコンポーネント

Pulldown menu component

2つのモードで動作:

  • SelectedValue: 選択した値がヘッダーに表示される（言語選択など）
  • FixedHeader: 固定のヘッダー名が表示される（メニューバーなど）
