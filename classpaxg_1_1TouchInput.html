Detailed Description

Multi-touch input abstraction class with state tracking.

状態追跡機能付きマルチタッチ入力抽象化クラス

Provides a platform-independent interface for retrieving multi-touch input data. Currently supports Android via DxLib, with null implementation for other platforms. Also tracks previous frame's touch state for click detection.
