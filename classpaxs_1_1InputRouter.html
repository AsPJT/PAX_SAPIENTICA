Detailed Description

統合入力ルーター

Unified input router

IInputHandler オブジェクトを登録し、レイヤー順序（前面→背面）で全ての入力イベントをルーティングします。 前面のハンドラーが入力を処理した場合、背面のハンドラーには伝播しません。
