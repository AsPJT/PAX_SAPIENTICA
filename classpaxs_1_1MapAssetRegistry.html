Detailed Description

地図アイコンアセット（テクスチャ）を統合管理するレジストリ

Unified registry for managing map icon assets (textures)

人物アイコン（Portraits）と地図アイコン（MapIcons）の2種類のテクスチャを管理し、 ハッシュキーによる高速アクセスを提供
